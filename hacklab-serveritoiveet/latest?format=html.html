Helsinki Hacklab list for server requests / serveritoiveiden lista

Please add your requests / needs for server capacity and functionality to this list
Lisää tähän listaan toiveesi / tarpeesi serverikapasiteetista ja -toiminnoista

I'd request you add your (nick)name next to your request or an existing one that you'd wish to see implemented. This is done to simplify mapping the demand for the requests and to help coordinate the implementation with the requestors. Feel free to add comments to requests - an easy way to help get special requests done. :)
Toivoisin että  listaan lisättäisiin myös toivojien nimet / nimimerkit, että tarpeet on helpompi  hoitaa tarvitsijoiden kanssa yhteisesti. Tarvekartoituksen edistämiseksi  lisääthän jo listassa oleviinkin kohtiin nimesi tarpeen mukaan.  Perusteluja saa toki myös lisätä ja niillä on helppo edistää  harvinaisempia toiveita! :)
