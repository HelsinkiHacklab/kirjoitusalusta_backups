if ja case rakenteet

wire     [3:0]    bcd;
reg       [6:0]    segments;

Pitää olla always-rakenteiden sisällä ja sijoitusten kohteiden pitää olla tyyppiä reg sekä kombinaatio- että sekvenssilogiikassa.

Voi ketjuttaa, syntetisoi yleensä prioriteettienkooderin:

if (bcd == 4'd0)
segments = 7'b1111110;
else if (bcd == 4'd1)
segments = 7'b1100000;
else
segments = 7'b0000000

// Sisältää virheitä, harjoitustehtävä
// Jos kaikki vaihtoehdot määritelty, defaultin käyttäminen helpottaa, syntetisoituu yleensä hakutauluksi (LUT) tai valitsimeksi (MUX)

case (bcd)
4'd0:         segments = 7'b1111110;
4'd1:         segments = 7'b0110000;
4'd2:         segments = 7'b1101101;
4'd3:         segments = 7'b1111001;
4'd4:         segments = 7'b0110011;
4'd5:         segments = 7'b1011011;
4'd6:         segments = 7'b1011111;
4'd7:         segments = 7'b0000000;
4'd8:         segments = 7'b0000000;
4'd9:         segments = 7'b0000100;
default :     segments = 7'b0111000;  // 4'hf
endcase
