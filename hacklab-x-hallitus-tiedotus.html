(lähetetty 26.8.2015 virallisella postilistalla)

Repair work, IRC network and Mini Maker Faire

* Kiinteistössä korjaustöitä / Building repairs *

Kiinteistössä suoritetaan korjaustöitä. Varastosta on varattu niitä varten työskentelytilaa ja veden käytössä voi olla rajoituksia, tarkempaa tietoa päivän tilanteesta IRCitse tai läbin seinältä löytyvistä tiedotteista. Arvioitu valmistuminen 4. syyskuuta.

There is repair work being done in the building this week, this requires some working-space in our storage and there may be limits to using water. More information about the situation each day is available via IRC or on notifications posted on the lab whiteboard. The work is estimated to be finished by Sept 4th.

* Siirtyminen Freenode-verkkoon / Moving to Freenode network *

Helsinki Hacklabin irc-kanava on siirretty IRCnetistä Freenoden puolelle. Vanhalla kanavalla voi kysellä neuvoja Freenodelle siirtymiseen. Verkon vaihtamisen taustoja on selitetty täällä:
http://helsinki.hacklab.fi/2015/08/siirtyminen-freenode-verkkoon/

Helsinki Hacklab irc channel has been moved from IRCnet to Freenode network. Ask for instructions on our old IRC channel how to move to Freenode.

* Espoo Mini Maker Faire *

Espoossa järjestetään Suomen ensimmäinen Mini Maker Faire 17.-18. lokakuuta. Tapahtuman järjestää Wärk:festien taustaltakin löytyvä Wärk ry. Hacklabit osallistuvat omalla pöydällään yhteisen hacklab.fi-nimikkeen alla.

Finlands first Mini Maker Faire will be held in Espoo on 17th – 18th of October. The event is held by Wärk ry, which was also behind previous Wärk:fest events. Hacklabs will take part in the event with a common hacklab.fi table.
