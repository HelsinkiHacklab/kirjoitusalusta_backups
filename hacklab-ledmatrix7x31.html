7x31 led matrix

Because that's what Styrge built, now we recycle the frame for something more usefull. It will be treated as 8x32 for simplified control, the last row&column just do not actually do anything.

Phase 1: single color, no PWM

Call it proof-of-concept, will be wired with cheap-ass leds just to test the ideas we have.

Rows are handled by NPN-transistors, columns by shift-register boards originally designed for the infamour reactor simulator (each board has 4 595 regs and resistors for the leds so 32 bits per board). https://github.com/HelsinkiHacklab/reactor/blob/master/blueprints/shift_reg.brd

If zzorn manages to wire up all the leds so that the columns can just be connected with IDC-connectors to the shift-reg board I (rambo) can get rest of this working in couple of hours.

Phase 2: PWM:d RGB

The fun part...

Wired so that we have high-power (2A) PNP transistor feeding the Common-Anode of a row
and 3x4 shift-registers to sink the colors (again same board as above, resistor values as per color).

Driving this will be interesting, we'll be using XMegas and leverage the DMA, suovula and zzorn have some idea of how to keep calculating the bit-patterns for PWM in between the DMA pushing data to the shift-regs. 

This is not only quite insane amount of soldering quite challenging code-wise as well...
